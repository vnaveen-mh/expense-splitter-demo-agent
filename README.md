# Expense Splitter Demo Agent

Demo agent built using ADK to demonstrate expense-splitter MCP tool

## Usage

1. Set `GEMINI_API_KEY` in your environment.
2. Build the MCP binary from https://github.com/vnaveen-mh/expense-splitter.
3. Copy the binary somewhere, e.g. `/tmp/expense-splitter`.
4. Run `go run .` (or pass `-mcp-bin` to point at the binary).
